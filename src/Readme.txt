
. All files in "data" directory needs to be in the programs directory when executing the program
. Details on instructions to execute the program is in the final report

. An overview of the commands is below
1) manual entry of data (with world point file "worldPt_manual.txt" already entered in the format mentioned in final report)
run =>	Main.py 2 worldPt_manual.txt

2)run non noisy data
run => Main.py 3 worldPt.txt imagePt.txt

3)run 1 noisy data
run => Main.py 3 noise_one_worldPt.txt noise_one_imagePt.txt

4)run 1 noisy data
run => Main.py 3 noise_two_worldPt.txt noise_two_imagePt.txt
